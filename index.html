Como Cadastrar uma Empresa (Escritório)

Siga este guia passo a passo para registrar um novo escritório no Ploomes.

Passo 1 Navegação inicial

Acessar Novo Cliente → Empresa

Para começar o cadastro de um novo escritório:

  1. No menu lateral, localize e clique na seção "Clientes"
  2. No canto superior direito, clique no botão "Novo Cliente"
  3. Selecione a opção "Empresa" no menu que aparecer

💡 Dica: A opção "Empresa" é usada especificamente para cadastrar escritórios de advocacia.

Passo 2 Preenchimento de dados

Preencher Informações do Escritório

No formulário que abrir, preencha os seguintes campos obrigatórios:

Nome: Digite o nome fantasia do escritório
Razão Social: Insira a razão social oficial
Pessoa Física: Selecione "Sim" ou "Não"
→ Se "Sim", preencha também o campo CPF
Marcadores: Adicione o marcador "Escritório"

⚠️ Importante: O marcador "Escritório" é essencial para identificar e filtrar escritórios no sistema.

Passo 3 Finalização

Salvar e Confirmar

Após preencher todos os campos:

  1. Revise os dados inseridos
  2. Clique no botão "Salvar"
  3. Aguarde o processamento

Você será redirecionado automaticamente para o perfil do escritório recém-criado, onde poderá visualizar e editar informações adicionais.

✅ Próximo passo: Com o escritório cadastrado, você já pode vincular advogados a ele!

Como Cadastrar um Advogado (Pessoa)

Aprenda a vincular advogados aos escritórios cadastrados no sistema.

Passo 1 Navegação inicial

Acessar Aba "Pessoas"

Para adicionar um advogado ao escritório:

  1. Acesse o perfil do escritório previamente cadastrado
  2. Localize e clique na aba "Pessoas" no menu do perfil

💡 Dica: Esta aba gerencia todos os contatos vinculados ao escritório.

Passo 2 Iniciar cadastro

Clicar em "Nova Pessoa"

Para iniciar o cadastro do advogado:

  • Clique no botão azul "Nova pessoa"
  • Um formulário de cadastro será aberto
Passo 3 Seleção de tipo

Selecionar Marcador "Advogado"

No formulário "Nova pessoa":

  1. Localize o campo "Marcadores"
  2. Selecione o marcador "Advogado"
  3. Os campos específicos (como OAB) aparecerão automaticamente

⚠️ Importante: Este marcador determina quais campos serão exibidos no formulário.

Passo 4 Preenchimento

Preencher Dados do Advogado

Preencha os campos obrigatórios:

Nome: Nome completo do advogado
OAB: Formato UFXXXXX
→ Exemplo: SP133996
Empresa: Preenchido automaticamente com o escritório atual
Passo 5 Finalização

Confirmação e Vínculo

Após preencher todos os campos:

  1. Revise os dados inseridos
  2. Clique no botão "Salvar"
  3. Aguarde o processamento

Você será redirecionado automaticamente para o perfil do advogado. O campo "Empresa" mostrará o vínculo com o escritório.

✅ Sucesso: O advogado está agora vinculado ao escritório!

Como Editar Informações do Reclamante

Aprenda a atualizar dados cadastrais, marcadores e telefones de clientes.

Passo 1 Navegação inicial

Acessar Menu de Edição

Para modificar os dados de um reclamante:

  1. Acesse o perfil do cliente que deseja editar
  2. Clique no botão "Opções" no canto superior
  3. Selecione "Editar Cliente" no menu dropdown
Passo 2 Verificação de dados

Verificar Marcador e CPF

No formulário de edição:

Marcador: Confirme que está marcado como "Reclamante"
CPF: Preencha o documento se estiver vazio

💡 Dica: O marcador correto garante que o cliente apareça nas listas filtradas.

Passo 3 Configuração de telefone

Selecionar Máscara do Brasil

Para configurar o formato correto do telefone:

  1. Localize a seção "Telefones"
  2. Clique no ícone à esquerda do campo (geralmente um globo ou bandeira)
  3. Selecione "+55 Brasil" na lista

⚠️ Importante: A máscara correta permite validação automática do formato.

Passo 4 Adicionar/remover números

Gerenciar Números de Telefone

Para gerenciar os telefones do cliente:

Adicionar: Clique no ícone "+" para inserir novos números
Remover: Clique no ícone "-" para excluir números
Formato: Digite (DDD) XXXXX-XXXX
Passo 5 Finalização

Confirmar e Salvar

Para aplicar as modificações:

  1. Revise todas as alterações realizadas
  2. Role até o final do formulário
  3. Clique no botão "Salvar"

✅ Sucesso: As alterações serão aplicadas imediatamente ao perfil!

Escritórios e Advogados no Padrão CNA/CNSA

Aprenda a consultar e verificar escritórios e advogados nos sistemas CNA e CNSA da OAB.

Passo 1 Acesso ao sistema

Acessar o Site do CNSA

Para consultar um escritório cadastrado na OAB:

  1. Acesse o site oficial: cnsa.oab.org.br
  2. Aguarde o carregamento completo da página

💡 Dica: O CNSA é o Cadastro Nacional de Sociedades de Advogados da OAB.

Passo 2 Busca

Realizar a Consulta

Para buscar o escritório:

  1. Preencha o campo "Nome" com o nome do escritório
  2. Complete a verificação "Não sou um robô"
  3. Clique em "Consultar"
  4. Aguarde os resultados aparecerem na tela

💡 Dica: Para encontrar o CNPJ do escritório e cadastrar na Ploomes, use o Empresa Dois ou o Google.

Passo 3 Verificação final

Verificar Advogados Associados

Para consultar os advogados vinculados ao escritório:

  1. Clique no resultado do escritório na lista
  2. Visualize os advogados associados
  3. Clique no nome de um advogado para ver mais detalhes
  4. Você será redirecionado ao perfil no CNA
  5. No perfil, consulte o número da OAB

✅ Importante: Sempre verifique a OAB no CNA para garantir dados corretos no cadastro.

Como Criar e Gerenciar Marcadores

Aprenda a criar e configurar marcadores para organizar seus negócios.

Passo 1 Navegação

Acessar Configuração de Marcadores

Para acessar as configurações:

  1. Na tela de "Negócios" (visão de funil)
  2. Clique no ícone de engrenagem (Menu) no canto superior direito
  3. Selecione "Configurar marcadores de negócios"
Passo 2 Gerenciamento

Editar ou Criar Marcadores

Para editar marcadores existentes:

  • Clique no marcador desejado
  • Altere o nome ou a cor

Para criar um novo marcador:

  1. Preencha o campo "Descrição" (nome do marcador)
  2. Escolha uma "Cor"
  3. Clique em "Adicionar"

💡 Dica: Use cores diferentes para categorias específicas e facilite a identificação visual dos negócios.

Como Criar Filtros Avançados

Crie filtros personalizados para encontrar negócios específicos rapidamente.

Passo 1 Iniciar

Acessar Menu de Filtros

Para criar um novo filtro:

  1. Na tela de "Negócios" (ou qualquer lista)
  2. Clique em "Filtros"
  3. No menu que abrir, clique em "Novo Filtro"
Passo 2 Configurações

Configuração Inicial e Visibilidade

No menu de criação:

  1. Navegue até a aba "Filtros Avançados"
  2. Dê um nome ao seu filtro
  3. Para compartilhar com a equipe, clique em "Visível para: Apenas eu"
  4. Selecione "Um ou mais usuários" → "Todos da Empresa"

💡 Dica: Compartilhar filtros com a equipe garante que todos trabalhem com os mesmos critérios.

Passo 3 Lógica do filtro

Adicionar Condições (A Lógica)

Configure a lógica do seu filtro na tabela:

  1. Campo: O que você quer filtrar (ex: "Marcadores")
  2. Operação: A lógica aplicada (ex: "Igual a" para incluir, "Diferente de" para excluir)
  3. Valor: O valor específico (ex: marcador "Escritório")
  4. Clique no ícone (+) para adicionar a condição

💡 Dica Avançada: Para filtrar por campos de outras entidades (ex: Reclamante), pesquise pelo campo "Cliente". Clique no botão "Cliente" que aparecer e o sistema mostrará todos os campos do Reclamante para você filtrar.

Guia em Vídeo (Passo 3)

Este passo é complexo. Assista ao vídeo curto abaixo mostrando o processo em detalhe.

Passo 4 Lógica avançada

Filtros Complexos (E / OU)

Para criar filtros mais sofisticados:

  1. Repita o Passo 3 para adicionar mais condições
  2. Use o botão "E" quando TODAS as condições devem ser verdadeiras
  3. Use o botão "OU" quando PELO MENOS UMA condição deve ser verdadeira

Exemplo: Marcador = "Escritório" E CPF = "Preenchido"

💡 Dica: Combine operadores E/OU para criar lógicas poderosas e encontrar exatamente o que precisa.

Passo 5 Finalização

Aplicar ou Salvar

Com o filtro pronto, escolha uma opção:

🔵 Aplicar

Usa o filtro imediatamente na tela atual, mas não o salva para uso futuro.

💾 Salvar

Salva o filtro na sua lista pessoal ou compartilhada para ser reutilizado sempre que precisar.

✅ Recomendação: Use "Salvar" para filtros que você usará frequentemente!

Como Cadastrar Leads B2B (Importação)

Aprenda a importar múltiplos leads em massa usando planilhas Excel.

Passo 1 Navegação inicial

Acessar Negócios e Selecionar Funil

Para começar a importação em massa:

  1. No menu lateral, clique em "Negócios"
  2. Na tela que abrir, selecione o funil onde deseja importar os leads

💡 Dica: Verifique se está no funil correto antes de prosseguir com a importação.

Passo 2 Configurações

Acessar Menu de Importação

Para acessar a ferramenta de importação:

  1. No canto superior direito, clique no ícone de engrenagem (Menu)
  2. No menu que abrir, selecione "Importar negócios pelo Excel"
Passo 3 Arquivo Excel

Fazer Upload da Planilha

Para enviar sua planilha:

  1. Clique em "Fazer upload da planilha"
  2. Selecione o arquivo Excel (.xlsx) do seu computador
  3. Aguarde o upload ser concluído

💡 Dica: Use o modelo de planilha abaixo para garantir que todos os campos estejam corretos.

📥 Baixar Modelo de Planilha
Passo 4 Configuração dos campos

Mapear os Campos

Para vincular os campos da planilha aos campos do Ploomes:

  1. Clique em cada caixa de seleção
  2. Escolha o campo correspondente do Ploomes
  3. Para definir em que estágio do funil os leads serão criados, mapeie o campo "Negócio - Estágio" usando o mesmo nome de estágio do Ploomes
  4. Repita para todos os campos da planilha

⚠️ Importante: No campo Responsável, preencha com seu nome de usuário do Ploomes.

Passo 5 Conclusão

Confirmar e Finalizar Importação

Para concluir o processo de importação:

  1. Revise todos os mapeamentos realizados
  2. Clique em "Confirmar"
  3. Aguarde a barra de progresso completar
  4. Clique em "Finalizar Importação"

✅ Sucesso: Seus leads foram importados e estão prontos para serem trabalhados no funil!

Como Cadastrar Leads B2B (Manual)

Aprenda a cadastrar negócios B2B manualmente no Ploomes.

Passo 1 Preparação

Certificar Entidades Criadas

Antes de criar negócios manualmente no funil, certifique-se de que:

  1. O escritório já está cadastrado no sistema
  2. Os advogados vinculados ao escritório estão cadastrados

Siga os tutoriais para cadastrar Escritório e Advogados se necessário.

💡 Dica: Esses cadastros são pré-requisitos essenciais para criar negócios B2B de forma eficiente.

Passo 2 Iniciar cadastro

Criar Novo Negócio

Para iniciar o cadastro manual:

  1. Localize o botão "Novo Negócio" no canto superior direito
  2. Clique no botão para abrir o formulário de criação
Passo 3 Configuração

Preencher Dados do Negócio

No formulário de criação:

  1. Selecione o estágio onde o negócio será criado
  2. Preencha o campo "Cliente" com o escritório
  3. O campo "Contato" será preenchido automaticamente com os advogados vinculados
  4. Para alterar o responsável, clique em "Outras Informações" e selecione seu usuário

💡 Dica: Certifique-se de que o escritório e advogados estão corretamente vinculados para que os campos sejam preenchidos automaticamente.

Passo 4 Conclusão

Confirmação e Conclusão

Para finalizar:

  1. Revise todas as informações preenchidas
  2. Clique em "Salvar" ou "Criar"
  3. Você será redirecionado automaticamente ao perfil do negócio

✅ Sucesso: Seu negócio B2B foi criado e está pronto para ser trabalhado!

Advogados Padrão CNA

Aprenda a consultar advogados cadastrados na Ordem dos Advogados do Brasil (OAB) através do sistema CNA.

Passo 1 Acesso ao sistema

Acessar o Site do CNA

Para consultar um advogado cadastrado na OAB:

  1. Acesse o site oficial: cna.oab.org.br
  2. Aguarde o carregamento completo da página
2

Realizar Busca do Advogado

Utilize os campos de busca para localizar o advogado desejado no sistema.

• Preencha o campo "Nome" com o nome completo do advogado
• Preencha o campo "N° da Inscrição" com o número da OAB
• Clique em "Pesquisar" para iniciar a busca
• Aguarde os resultados aparecerem na tela

Dica Importante

Caso não encontre o advogado, tente limpar o número da OAB de qualquer zero à esquerda. Por exemplo: mude "012345" para "12345".

3

Verificar Perfil do Advogado

Ao clicar em algum resultado da busca, você terá acesso ao perfil completo do advogado.

• Visualize todas as informações profissionais do advogado
• Verifique as sociedades/escritórios aos quais está vinculado
• Confirme o status atual da inscrição na OAB

Atenção

Se o advogado não possui nenhuma sociedade vinculada, devemos cadastrar o escritório como "Pessoa física - Sim". Revisite o tutorial de criação de escritório para mais detalhes.

4

Consultar Sociedades Vinculadas

Para acessar informações completas sobre os escritórios aos quais o advogado está vinculado.

• Clique na aba "Sociedade" no perfil do advogado
• Você poderá ver todas as sociedades associadas ao advogado
• Ao clicar em alguma sociedade, você será redirecionado ao perfil dela no CNSA

Benefício

Esta consulta permite verificar todas as informações do escritório diretamente no sistema oficial da OAB.

Realizar Solicitação

Aprenda a criar e enviar solicitações de importação de dados no Jestor.

Passo 1 Acesso ao sistema

Abrir Menu de Solicitação

Para iniciar uma nova importação:

  1. Localize o menu superior esquerdo
  2. Clique na aba "Solicitar"
  3. Aguarde o formulário carregar

ℹ️ Nota: É necessário ter uma conta ativa no Jestor. Caso tenha problemas de acesso, contate o administrador.

Passo 2 Dados Básicos

Preencher o Formulário

Configure os detalhes do pedido preenchendo:

  • O campo Time
  • A identificação da Mesa
  • O Tipo de requisição correto (Enriquecimento + Subida Ploomes)

💡 Padronização: Utilize nosso modelo oficial para evitar erros na leitura dos dados.

Baixar Planilha Modelo
Passo 3 Preparar Arquivo

Preencher a Planilha

Copie os dados para o modelo baixado seguindo as regras:

  1. Defina o Estágio (ex: "NOVO LEADS JPA")
  2. Use a coluna Marcadores para adicionar tags ao lead
  3. Para marcadores extras, crie colunas com prefixo "Marcadores..."

⚠️ Atenção: O nome do estágio na planilha deve ser idêntico ao do funil no Ploomes.

⚠️ Importante: Não use a descrição para definir estágio do funil ou marcadores. Para melhor prática, utilize as colunas da planilha Excel.
